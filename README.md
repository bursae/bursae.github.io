# Site Workflow

## Add A New Post
1. Copy `posts/_template.md` to a new file in `posts/`.
2. Edit the title, image, and body.
3. Add a new entry at the top of `posts/posts.json`.
4. Keep dates in `YYYY` format.

## Preview Locally
1. Run `./scripts/serve-local.sh`.
2. Open `http://localhost:8000/site/notes.html`.

## Publish
1. Commit your changes.
2. Push to GitHub.
